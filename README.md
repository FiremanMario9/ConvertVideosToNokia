# ConvertVideosToNokia
I understand that not much documentation has been put into the nokia 225 4g, so this guide will help you convert videos to it.

Presiquites:
1. An SD card for storing videos.
2. A PC with FFMPEG installed.
3. A video file. I reccommend Nzuri Video Downloader [https://nzurivideodownloader.vercel.app/] to download YouTube Videos, or just use any movies/videos you already have.

Conversion:
1. I did the hard work. Now all you need to do is download the .bat file.
2. Drag and drop your video (.webm, .mp4, .mkv are supported) onto the .bat file.
3. If everything is sucsessful (you have ffmpeg installed) a cmd prompt should pop up. A time will appear at the bottom of the command prompt. That is the time in the video it is currently converting. Once finished, it will close.
4. The file will appear as yourvideoname.oldextention.flv
5. If you wish, delete the old extention name. .flv is the correct extention.
6. Copy the video file to your nokia's "Video" folder. Make sure it is the SD card, not the internal storage. The internal storage is VERY limited, it only has 32 MB. Don't expect to copy anything of size to it. Even a short video will take up ALL of the internal storage.
7. Open the videos app on the nokia and click the video! Alternativly, open the files app and go to the "Video" folder, and click on the file there.
